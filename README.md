# Otob-sBilet.Rs

University project for the Object-Oriented Programming (OOP) course.
This is a simple bus ticket reservation app built with C# and WinForms.

## Course Context
- Built for a university OOP course.
- Uses interfaces covered in class (see `IRezervasyon`).
- Applies core OOP concepts: abstraction, inheritance, and encapsulation.

## Features
- Create a trip (sefer) with origin, destination, and date/time.
- Choose seats and reserve them.
- Print or save ticket info.

## Project Structure
- `Sefer`, `SehiriciSefer`, `SehirlerarasiSefer`: trip model.
- `Otobus`, `Koltuk`: bus and seat model.
- `IRezervasyon`: reservation interface used in the course.
- `MainForm` + user controls: WinForms UI.

## How To Run
1. Open the solution `Otob-sBilet.R.S.sln` in Visual Studio.
2. Build and run the project.

## Notes
- This is an educational project for the OOP course.
- UI text is kept simple and ASCII-only for compatibility.
